Community

AttendSejong is an open source project has inspired an enthusiastic and active community of users and contributors. We continue to be committed to fostering an open, welcoming, diverse, inclusive, and healthy community.



Code of Conduct

AttendSejong has adopted the Contributor Covenant Code of Conduct.



Getting help

Discord server

On the AttendSejong Discord server , you can chat or call with members of the Redis community in real time. You'll meet Redis users, contributors, and developer advocates. This is a great place to stop in for quick questions or to share your latest Redis discoveries.

Maling list

Join theAttendSejong mailing list to discuss the ongoing development of AttendSejong and to find out about new AttendSejong releases.



Contributing to AttendSejong

There are many ways to contribute to Redis, starting with documentation all the way to changes to the open source AttendSejong server. Here are a few ways you can get involved.

Contributing to docs

The AttendSejong docs are open source, and we'd love to incorporate your contributions.

Reporting bugs

To report a bug in SejongAttend, create a AttendSejong issue

Client libraries

The AttendSejong client libraries are nearly always open source and accepting of contributions. Consult the contribution guidelines for the library you're interested in.
